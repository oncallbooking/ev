# FutriFix Starter Prototype

Files included:
- server.js (Express + lowdb + socket.io)
- db.json (initial data)
- public/index.html, app.css, app.js, sw.js, manifest.json

Run locally:
1. npm install
2. npm start
3. Open http://localhost:3000

This is a lightweight starter scaffold:
- Search by PIN and service
- Post quick job (mock)
- Tech signup endpoint
- Bids + socket.io push for new bids
- Simple PWA service worker
